# 🧛‍♂️ Wlademir Bizarre Adventures

Um **Dungeon Crawler Procedural** em Pixel Art (32x32) feito na **Godot 4**.

## 📖 A História
Wlademir é um vampiro explorando ruínas ancestrais no Egito em busca da cura contra o sol. O labirinto é gerado proceduralmente a cada partida, e o sol se move em ciclos de dia e noite, tornando cada sala um desafio de luz e sombra.

---

## 🎨 Guia para o Artista
Olá! Para mantermos o projeto organizado e funcional, siga estas regras:

### 📁 Onde colocar seus arquivos:
- **Personagens e Objetos**: `assets/sprites/`
- **Sons e Músicas**: `assets/sounds/`

### 📏 Padrões Técnicos:
- **Resolução Nativa**: 320x180 pixels.
- **Tamanho dos Tiles/Personagens**: Base de **32x32** pixels.
- **Formato**: Utilize `.png` com fundo transparente ou `.svg`.

---

## 🛠️ Como Contribuir (GitHub)
Para evitar conflitos de versão, siga este fluxo:

1. **Antes de começar**: Sempre dê um `Pull` (Puxar) no GitHub Desktop para baixar as últimas mudanças do programador.
2. **Ao terminar**: Salve suas artes na pasta correta dentro do projeto.
3. **Commit**: Escreva o que você adicionou (ex: "Adicionado sprite do caixão").
4. **Push**: Envie suas mudanças para o servidor.

---

## 🚀 Como Rodar o Projeto
1. Instale a [Godot Engine 4.x](https://godotengine.org/).
2. Clone este repositório no seu computador.
3. No Godot Project Manager, clique em **Import** e selecione o arquivo `project.godot`.
4. Aperte **F5** para rodar a cena principal (`scenes/level_generator.tscn`).

---

## 🎮 Controles Atuais
- **Setas / WASD**: Movimentação.
- **Empurrar**: Basta caminhar em direção às caixas.
- **F11**: Alternar Tela Cheia (Fullscreen).
- **F5**: Reiniciar o labirinto.
