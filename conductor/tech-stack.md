# Tech Stack

## Game Engine
- **Godot Engine 4 (4.6):** The primary development environment used for building the game, handling rendering, physics, and scene management.

## Programming Language
- **GDScript:** The native scripting language used for all game logic, mechanics, and procedural generation.

## Export Targets
- **Web (HTML5/WebAssembly):** The game is configured for web deployment, ensuring accessibility directly through the browser.
- **Desktop:** Standard executable export for PC.
